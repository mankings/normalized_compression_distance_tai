## How to run

```make clean && make all```

```./bin/shazam [ -c compressor] [ -d dataset ] -f sampleFile ```

| Option                                         | Description                    |
|------------------------------------------------|--------------------------------|
| -h                                             | This help                      |
| -v                                             | Verbose mode                   |
| -c [GZIP, BZIP2, LZMA, ZSTD, LZ4, LZO, SNAPPY] | Compressor type (defautl: GZIP)|
| -d (folder)                                    | Dataset folder                 |
| -f (file)                                      | Sample file                    |
